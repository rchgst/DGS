# Documentación del Proyecto: AlgoBlocks

Esta carpeta contiene toda la documentación del proyecto estructurada de la siguiente manera:

* **[arquitectura/](./arquitectura/)**: Documentos de arquitectura de software, decisiones de diseño técnico y especificación del stack.
* **[casos_de_uso/](./casos_de_uso/)**: Casos de uso detallados para alumnos, profesores y administradores.
* **[wireframes/](./wireframes/)**: Prototipos y bocetos de la interfaz de usuario.
* **[diagramas/](./diagramas/)**: Diagramas de flujo, diagramas entidad-relación (ERD) de base de datos y diagramas de arquitectura.
* **[api/](./api/)**: Documentación de los endpoints de la API, payloads y códigos de estado.
* **[reuniones/](./reuniones/)**: Registro y minutas de las reuniones del equipo de desarrollo.
